# Exercise 3

I chose to implement this exercise using React.
Hosted at: [www.genuinejon.com](http://www.genuinejon.com:8081)


## Install Dependencies
```
npm install
```

## Run Tests
```
npm test
```

## Build
This will put an optimized build in the ./build directory

```
npm run build
```


## Serve
```
npm install -g serve
serve -s build -p 80
```
